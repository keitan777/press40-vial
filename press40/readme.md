# 3dango_academy/press40

*A short description of the keyboard/project*

* Keyboard Maintainer: [3dango-academy](https://github.com/3dango-academy)

Make example for this keyboard (after setting up your build environment):

    make 3dango_academy/press40:default

Copy the created uf2 file to the RPI_RP2 drive.

## Bootloader
* **Physical reset button**: Press the button on the PCB's reverse side while connecting the USB cable.
* **Keycode in layout**: Press the key mapped to `QK_BOOT` if it is available
